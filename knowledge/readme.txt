================================================================================
                          README.md 内容规范
================================================================================

【文件性质】
- 项目说明文档，主要给人看
- 也可以给 AI 参考，但不会自动加载
- 通常随项目提交到 Git 仓库

【应该包含的内容】
1. 项目名称和简介
2. 功能特性列表
3. 技术栈
4. 环境要求
5. 安装步骤
6. 使用说明
7. 项目结构（可选）
8. 贡献指南（可选，开源项目需要）
9. License
